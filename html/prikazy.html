Linux Command Reference

ls - výpis souborů

$ ls -la

Příkaz ls zobrazí obsah adresáře.

cd - změna adresáře

$ cd /cesta/k/adresari

Příkaz cd změní aktuální pracovní adresář.

userdel - odstranění uživatele

$ sudo userdel uzivatel

Příkaz userdel odstraní uživatele ze systému.

passwd - změna hesla

$ passwd

Příkaz passwd změní heslo aktuálního uživatele.

ifconfig - síťová rozhraní

$ ifconfig

Příkaz ifconfig zobrazí informace o síťových rozhraních.

iwconfig - bezdrátová rozhraní

$ iwconfig

Příkaz iwconfig zobrazí informace o bezdrátových rozhraních.

route - směrovací tabulka

$ route -n

Příkaz route zobrazí směrovací tabulku.

service - správa služeb

$ sudo service apache2 restart

Příkaz service spravuje systémové služby.

systemctl - správa systémových služeb

$ sudo systemctl status sshd

Příkaz systemctl spravuje systémové služby v systemd.

crontab - plánování úloh

$ crontab -e

Příkaz crontab umožňuje plánování pravidelných úloh.

alias - vytvoření aliasu

$ alias ll='ls -la'

Příkaz alias vytvoří zkratku pro příkazy.

unzip - rozbalení archivu

$ unzip archiv.zip

Příkaz unzip rozbalí zip archiv.

touch - vytvoření souboru

$ touch novy_soubor.txt

Příkaz touch vytvoří nový prázdný soubor nebo aktualizuje časové razítko existujícího souboru.

mkdir - vytvoření adresáře

$ mkdir novy_adresar

Příkaz mkdir vytvoří nový adresář.

nano - textový editor

$ nano soubor.txt

Příkaz nano otevře jednoduchý textový editor.

cat - zobrazení obsahu souboru

$ cat soubor.txt

Příkaz cat zobrazí obsah souboru.

scp - bezpečné kopírování

$ scp soubor.txt user@remote:/cesta

Příkaz scp bezpečně kopíruje soubory mezi počítači.

ssh - vzdálené připojení

$ ssh user@remote

Příkaz ssh umožňuje vzdálené připojení k jinému počítači.

rsync - synchronizace souborů

$ rsync -avz zdroj/ cil/

Příkaz rsync synchronizuje soubory a adresáře mezi lokálním a vzdáleným systémem.

diff - porovnání souborů

$ diff soubor1 soubor2

Příkaz diff porovnává dva soubory a zobrazuje rozdíly.

curl - stahování dat

$ curl -O https://example.com/soubor

Příkaz curl stahuje data z webových serverů.

ftp - přenos souborů

$ ftp hostname

Příkaz ftp umožňuje přenos souborů přes FTP protokol.

wc - počet řádků, slov a znaků

$ wc soubor.txt

Příkaz wc počítá řádky, slova a znaky v souboru.

head - zobrazení začátku souboru

$ head -n 10 soubor.txt

Příkaz head zobrazí prvních 10 řádků souboru.

tail - zobrazení konce souboru

$ tail -f logfile.log

Příkaz tail zobrazí posledních 10 řádků souboru a sleduje změny.

gzip - komprese souborů

$ gzip soubor

Příkaz gzip komprimuje soubory pomocí gzip algoritmu.

gunzip - dekomprese souborů

$ gunzip soubor.gz

Příkaz gunzip dekomprimuje soubory komprimované pomocí gzip.

tar - archivace souborů

$ tar -cvf archiv.tar slozka/

Příkaz tar vytváří archivy souborů.

untar - rozbalení archivu

$ tar -xvf archiv.tar

Příkaz tar rozbalí archiv souborů.

sed - stream editor

$ sed 's/staré/nové/' soubor.txt

Příkaz sed provádí textové transformace na vstupním streamu.

awk - textový procesor

$ awk '{print $1}' soubor.txt

Příkaz awk je mocný nástroj pro zpracování textu.

cut - výřez textu

$ cut -d: -f1 /etc/passwd

Příkaz cut vyřezává části řádků ze souborů.

tr - transformace znaků

$ tr 'a-z' 'A-Z' < soubor.txt

Příkaz tr transformuje nebo maže znaky.
